1. CONTACTFLOW - ⭐⭐⭐⭐⭐ (RECOMENDADO - MÁS COMPLETO)
================================================================================

🔗 Link de Repositorio:
   https://github.com/moelkenzi/contactFlow

📋 Descripción:
   Sistema versátil de gestión de contactos con interfaces GUI y CLI.
   Construido en Python con múltiples opciones de almacenamiento.
   ⭐ 9 stars | Actualizado recientemente

📌 ¿Qué es?
   Aplicación moderna que permite organizar conexiones de forma eficiente
   con opciones de almacenamiento flexible (JSON o SQLite).

✨ Características Principales:
   - Interfaz GUI moderna con soporte dark mode
   - Dual interface: GUI (gráfica) y CLI (línea de comandos)
   - Almacenamiento flexible: JSON o SQLite
   - Búsqueda avanzada de contactos
   - Seguimiento de fecha de creación
   - Agregar/eliminar contactos fácilmente
   - Auto-guardado y carga automática
   - Seguimiento del historial de contactos

📊 Datos que Maneja:
   - Nombre del contacto
   - Número de teléfono
   - Email
   - Notas adicionales
   - Fecha de creación

🛠️ Tecnologías Utilizadas:
   - Python (lenguaje principal)
   - Tkinter y CustomTkinter (para GUI)
   - JSON y SQLite (almacenamiento)

📁 Estructura del Proyecto:
   - GUI.py: Interfaz gráfica moderna
   - update.py: Interfaz de línea de comandos
   - sample.py: Implementación SQLite
   - contacts.json: Almacenamiento en JSON
   - contacts.db: Base de datos SQLite

🎯 Por qué es BUENO para tu informe:
   ✅ Estructura profesional y modular
   ✅ Múltiples opciones de almacenamiento
   ✅ Documentación clara
   ✅ Características avanzadas (búsqueda, historial)
   ✅ Manejo de errores comprensivo
   ✅ Licencia MIT (código abierto)

CÓMO USAR ESTE DOCUMENTO
================================================================================

1. PARA DESCARGAR LOS REPOSITORIOS:
   - Abre cada link en tu navegador
   - Click en el botón verde "Code"
   - Selecciona "Download ZIP"
   - O clona con: git clone [URL del repositorio]

2. PARA TU INFORME ISO:
   - Selecciona el repositorio que más te interese (recomendamos ContactFlow)
   - Analiza su estructura de clases
   - Evalúa cómo cumple con ISO 25010:
     * Funcionalidad
     * Confiabilidad
     * Usabilidad
     * Eficiencia
     * Mantenibilidad
     * Portabilidad
   - Evalúa cómo cumple con ISO 9001:
     * Documentación
     * Procesos definidos
     * Control de calidad
     * Mejora continua

3. PARA CLONAR Y TRABAJAR LOCALMENTE:
   git clone https://github.com/[USUARIO]/[REPOSITORIO].git
   cd [REPOSITORIO]
   python archivo_principal.py

CONTACTO Y REFERENCIAS

Para más información sobre las normas ISO:

ISO 25010 (Calidad del Software):
- Define características de calidad del software
- Enfoque en: Funcionalidad, Confiabilidad, Usabilidad, Eficiencia,
  Mantenibilidad, Portabilidad, Seguridad, Compatibilidad

ISO 9001 (Gestión de Calidad):
- Define requisitos para sistemas de gestión de calidad
- Enfoque en: Liderazgo, Planificación, Apoyo, Operación, Evaluación,
  Mejora
